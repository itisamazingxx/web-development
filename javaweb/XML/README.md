# XML
1. 可扩展标记语言
2. XML被设计用来传输和存储数据
3. Spring中的bean就是xml文件配置
4. 配置和业务逻辑分开是为了灵活 易于管理

### 什么情况下需要用到XML
1. 两个程序间进行数据通信(现在用json)
2. 配置文件(读取监听的端口号, 连接数据库的用户名和密码)

    tomcat服务器的server.xml, web.xml
3. 能存储复杂的数据关系

### 元素
1. 每个XML文件有且只有一个root
2. root的开始标记要在所有元素前
3. root的结束标记要在所有元素后
4. 元素, 节点, 标签是一个东西
